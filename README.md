### Run Ð¡ode

#### Get `started`

```bash
	git clone https://github.com/luamoris/webland.git
	cd webland
	npm i
	npm start
```

#### For `build`

```bash
	npm run dev 	// development
	npm run build	// production
```

#### For `lint`

> - [`stylelint`][stylelint]
> - [`eslint`][eslint]

```bash
	npm run lint		// lint Javascript files
	npm run stylelint	// lint SCSS files
```
